# 3) نطاق المنصة / Platform Scope

    - مبدأ النطاق
        - النطاق ليس “قائمة فئات” فقط، بل **طبقات تشغيل** يمكن إعادة استخدامها عبر الدول: كتالوج، مخزون، تسعير، طلبات، دفع، إسناد، جودة، نزاعات، امتثال.
        - لكل فئة نحدد: حساسية تشغيل، حساسية تنظيم، متطلبات جودة، ومتطلبات سلسلة الإمداد، ثم نُفعل الحزم المناسبة.
    - سلاسل التوريد (من المصدر إلى الإتمام)
        - الإنتاج الأولي
            - مزارع، مزارع مائية، مزارع دواجن، مصادر محلية.
            - قدرات مطلوبة: إدارة دفعات، تواريخ صلاحية، تتبع منشأ، مواصفات جودة.
        - التحويل والتصنيع
            - مصانع أغذية، مصانع مياه، مصانع منتجات استهلاكية.
            - قدرات مطلوبة: تخطيط توريد، عقود توريد، فحص جودة، توثيق اعتماد.
        - التخزين والإتمام
            - مستودعات، مراكز إتمام، تبريد وتجمد.
            - قدرات مطلوبة: سعات، نوافذ زمنية، جدولة تحميل وتنزيل، إدارة مواقع تخزين، لوجستيات عكسية.
        - النقل بيني
            - نقل بين المدن أو داخل المدينة بين المستودع والمتجر.
            - قدرات مطلوبة: تتبع شحنات، إثبات تسليم، إدارة حمولة، إدارة نوافذ.
    - الاستهلاك والتجارة والخدمات (آخر ميل وما بعده)
        - تجارة التجزئة اليومية
            - بقالة وتموين، منتجات منزلية، مستلزمات أطفال.
            - قدرات مطلوبة: بدائل، تجزئة طلب، جودة تعبئة، تتبع لحظي.
        - فئات حساسة ومنظمة
            - منتجات صحية وعناية، أدوية حسب الأنظمة، منتجات تتطلب عمرًا أو ضوابط.
            - قدرات مطلوبة: تحقق هوية، قيود تسليم، إثبات تسليم قوي، سجل تدقيق.
        - فئات تحتاج سلسلة تبريد
            - لحوم، أسماك، ألبان، آيس كريم.
            - قدرات مطلوبة: ضوابط تبريد، نوافذ زمنية دقيقة، مراقبة جودة.
        - خدمات حسب الطلب (غير مادية)
            - خدمات «أنجز لي»، خدمات منزلية وميدانية (صيانة، تركيب، إصلاح) وفق الأنظمة.
            - قدرات مطلوبة: جدولة مواعيد، تسعير خدمة، توزيع مهام، إثبات إنجاز، إدارة شكاوى.
        - تجارة بين الأفراد مع إتمام مساعد
            - بيع وشراء بين الأفراد مع ضمانات تشغيل.
            - قدرات مطلوبة: حجز مؤقت، إثبات تسليم، نزاعات، تقييمات، سياسات حماية.
        - خدمات موارد المدينة
            - مياه وغاز ووقود وطاقة حسب الأنظمة.
            - قدرات مطلوبة: امتثال صارم، جدولة، تحقق، وسياسات فئات.
    - أعمال إلى أعمال (تشغيل مؤسسي)
        - توريد مؤسسي
            - عقود توريد دورية للمنشآت (مطاعم، مقاهٍ، مدارس، مكاتب).
            - قدرات مطلوبة: اتفاقية مستوى الخدمة، فواتير دورية، شروط دفع، تسوية.
        - لوجستيات للمنشآت
            - نقل بيني + توصيل آخر ميل للمنشآت.
            - قدرات مطلوبة: نوافذ زمنية صارمة، جدولة تحميل وتنزيل، تتبع وإثبات تسليم.
        - تكاملات الشركاء
            - تكاملات نقاط البيع وإدارة المخزون وإدارة المستودعات وفق حجم الشريك.
            - قدرات مطلوبة: واجهات تكامل، خرائط بيانات، سجل تدقيق، إدارة تغييرات.
    - خريطة التوسع للفئات (نهج عالمي)
        - البداية: بقالة وتموين + خدمات محلية قليلة الحساسية.
        - التوسع: فئات طازجة وتبريد بعد جاهزية الجودة.
        - فئات منظمة: بعد جاهزية الامتثال والتحقق.
        - أعمال إلى أعمال: بعد ثبات التشغيل وتوفر شراكات قوية.
        - المعيار دائمًا: بوابات القرار (الاعتمادية، الاقتصاد، الثقة، الامتثال).
