# 11) حوكمة العمليات / Operations Governance

## الإجراءات التشغيلية القياسية الإلزامية / Mandatory SOPs

### 1. إجراءات التنفيذ والاستبدال / Fulfillment and Replacement

#### عملية الانتقاء / Picking Process
1. **استلام قائمة الانتقاء**: عبر تطبيق العامل
2. **التحقق من الموقع**: المنطقة الصحيحة في المستودع/المتجر
3. **انتقاء المنتجات**: حسب الترتيب الأمثل
4. **مسح الباركود**: لكل منتج للتحقق
5. **التحقق من الجودة**: فحص بصري للطازج والقابل للتلف
6. **التحقق من تاريخ الانتهاء**: FIFO (First In, First Out)

#### منطق الاستبدال / Substitution Logic
- **الأولوية الأولى**: نفس المنتج، حجم مختلف
- **الأولوية الثانية**: نفس العلامة التجارية، نوع مماثل
- **الأولوية الثالثة**: علامة تجارية مماثلة، نفس الفئة
- **القواعد**:
  - السعر يجب ألا يتجاوز السعر الأصلي بأكثر من 10%
  - الجودة يجب أن تكون مماثلة أو أفضل
  - موافقة العميل مطلوبة للاستبدالات الكبيرة

#### التعبئة / Packing
1. **الفصل**: منتجات مبردة/مجمدة/جافة منفصلة
2. **الحماية**: تغليف المنتجات القابلة للكسر
3. **الملصقات**: ملصق الطلب + ملصق العميل
4. **التحقق النهائي**: مطابقة المنتجات مع القائمة

### 2. إجراءات الإرجاع / Return Procedures

#### الأسباب المقبولة / Acceptable Reasons
- منتج تالف أو منتهي الصلاحية
- منتج خاطئ (ليس ما طلبه العميل)
- جودة غير مقبولة
- استبدال غير موافق عليه

#### العملية / Process
1. **الإبلاغ**: العميل يبلغ عن المشكلة (تطبيق أو دعم)
2. **التحقق**: السائق أو فريق الدعم يتحقق من الادعاء
3. **التوثيق**: صور للمنتج، سبب الإرجاع
4. **القرار**:
   - استرداد كامل
   - استبدال فوري (إن أمكن)
   - رصيد للطلب القادم
5. **التسجيل**: في النظام للتحليل

#### الإطار الزمني / Timeframe
- **الإبلاغ**: خلال 24 ساعة من التسليم
- **المعالجة**: خلال 48 ساعة
- **الاسترداد**: خلال 5-7 أيام عمل

### 3. إجراءات الاستدعاء / Recall Procedures

#### أنواع الاستدعاء / Recall Types
- **Class I**: خطر صحي جدي (فوري)
- **Class II**: خطر صحي محتمل (24 ساعة)
- **Class III**: مشكلة جودة بسيطة (3-5 أيام)

#### العملية / Process
1. **التنبيه**: إشعار من المورد أو الجهات التنظيمية
2. **التحديد**: تحديد جميع الوحدات المتأثرة في النظام
3. **الإيقاف**: إيقاف بيع المنتج فوراً
4. **الإشعار**: إبلاغ جميع العملاء الذين اشتروا المنتج
5. **الاسترجاع**: استرجاع المنتج من العملاء والمخزون
6. **الاسترداد**: استرداد كامل تلقائي
7. **التوثيق**: تقرير كامل للجهات التنظيمية
8. **المتابعة**: التأكد من استرجاع جميع الوحدات

### 4. إدارة الحوادث / Incident Management

#### تصنيف الحوادث / Incident Classification
- **P0 (حرج)**: توقف كامل للخدمة، تسرب بيانات
- **P1 (عالي)**: خدمة متدهورة بشكل كبير، مشاكل دفع
- **P2 (متوسط)**: مشاكل تؤثر على بعض المستخدمين
- **P3 (منخفض)**: مشاكل بسيطة، طلبات ميزات

#### عملية الاستجابة / Response Process
1. **الكشف**: من المراقبة أو التقارير
2. **التنبيه**: إشعار فريق الاستجابة
3. **التقييم**: تحديد الخطورة والتأثير
4. **التصعيد**: حسب مستوى الخطورة
5. **التخفيف**: إجراءات فورية لتقليل التأثير
6. **الحل**: إصلاح المشكلة الأساسية
7. **التواصل**: تحديثات منتظمة لأصحاب المصلحة
8. **التحليل البعدي**: Post-mortem بدون لوم

#### أهداف زمن الاستجابة / Response Time Targets
- **P0**: استجابة فورية، حل خلال 4 ساعات
- **P1**: استجابة خلال 15 دقيقة، حل خلال 24 ساعة
- **P2**: استجابة خلال ساعة، حل خلال أسبوع
- **P3**: استجابة خلال يوم عمل، حل حسب الأولوية

### 5. الإجراءات التصحيحية والوقائية / CAPA

#### متى تُطبق CAPA / When to Apply CAPA
- بعد أي حادث P0 أو P1
- عند اكتشاف نمط من المشاكل المتكررة
- بعد شكاوى العملاء الكبيرة
- عند انتهاكات الامتثال

#### عملية CAPA / CAPA Process
1. **تحديد المشكلة**: ما الذي حدث بالضبط؟
2. **تحليل السبب الجذري**: لماذا حدث؟ (5 Whys, Fishbone)
3. **الإجراءات التصحيحية**: إصلاح المشكلة الحالية
4. **الإجراءات الوقائية**: منع تكرارها
5. **التنفيذ**: تطبيق الإجراءات
6. **التحقق**: التأكد من فعالية الإجراءات
7. **التوثيق**: توثيق كامل للعملية والنتائج
8. **المراجعة**: مراجعة دورية للتأكد من الاستدامة

## حوكمة الشركاء / Partner Governance

### 1. درجات الجودة / Quality Grades

#### نظام التصنيف / Grading System
- **ممتاز (A)**: 90-100 نقطة
- **جيد جداً (B)**: 80-89 نقطة
- **جيد (C)**: 70-79 نقطة
- **مقبول (D)**: 60-69 نقطة
- **غير مقبول (F)**: <60 نقطة

#### معايير التقييم / Evaluation Criteria
- **دقة المخزون**: 25%
- **سرعة التنفيذ**: 20%
- **جودة المنتجات**: 25%
- **رضا العملاء**: 15%
- **الامتثال للسياسات**: 15%

#### التقييم / Assessment
- **تقييم أسبوعي**: حساب النقاط تلقائياً
- **مراجعة شهرية**: مع الشريك
- **مراجعة ربع سنوية**: شاملة مع خطة تحسين

### 2. عمليات التدقيق / Audits

#### أنواع التدقيق / Audit Types
- **تدقيق منتظم**: كل 3 أشهر لجميع الشركاء
- **تدقيق مفاجئ**: عند الاشتباه بمشاكل
- **تدقيق متابعة**: بعد وجود مشاكل

#### ما يُدقق / What is Audited
- **المخزون**: مطابقة الفعلي مع النظام
- **الجودة**: معايير التخزين والتعامل
- **النظافة والسلامة**: الامتثال لمعايير الصحة
- **العمليات**: الالتزام بالـ SOPs
- **التوثيق**: السجلات والوثائق

#### التقرير / Reporting
- تقرير تفصيلي خلال 3 أيام عمل
- درجة التدقيق (Pass/Fail/Conditional)
- خطة عمل للمشاكل المكتشفة

### 3. خطط التحسين / Improvement Plans

#### متى تُطلب / When Required
- درجة جودة C أو أقل
- فشل في التدقيق
- شكاوى العملاء >5% من الطلبات
- انتهاكات متكررة للسياسات

#### مكونات الخطة / Plan Components
1. **تحديد الفجوات**: المشاكل والأسباب
2. **الأهداف**: أهداف SMART قابلة للقياس
3. **الإجراءات**: خطوات محددة لكل هدف
4. **الجدول الزمني**: مواعيد محددة
5. **المسؤوليات**: من يفعل ماذا
6. **الموارد**: ما هو المطلوب
7. **القياس**: كيف نقيس النجاح

#### المتابعة / Follow-up
- اجتماعات أسبوعية للتقدم
- مراجعة شهرية للنتائج
- إعادة التقييم بعد 3 أشهر

#### العواقب / Consequences
- **لا تحسن**: تعليق الشريك
- **عدم الامتثال**: إنهاء الشراكة
- **تحسن ممتاز**: مكافآت وحوافز

## حوكمة القوى العاملة / Workforce Governance

### 1. حوافز الجودة / Quality Incentives

#### للعاملين في التنفيذ / For Fulfillment Workers
- **دقة الانتقاء**: مكافأة لـ >98% دقة
- **السرعة**: مكافأة للانتقاء في الوقت القياسي
- **عدم التلف**: مكافأة لعدم تلف المنتجات
- **تقييمات العملاء**: مكافأة للتقييمات الإيجابية

#### للسائقين / For Drivers
- **التسليم في الوقت المحدد**: مكافأة لـ >95% في الوقت
- **رضا العملاء**: مكافأة لتقييم >4.5/5
- **عدد التوصيلات**: مكافأة لتحقيق الأهداف
- **السلامة**: مكافأة لقيادة آمنة

### 2. سياسات الإلغاء والتأخير / Cancel/Late Policies

#### سياسة الإلغاء / Cancellation Policy

**للسائقين**:
- **إلغاء 1**: تحذير
- **إلغاء 2 في شهر**: إيقاف مؤقت 3 أيام
- **إلغاء 3 في شهر**: إيقاف مؤقت أسبوع
- **إلغاء 4+**: إنهاء التعاقد

**للعمال**:
- **غياب بدون عذر 1**: تحذير
- **غياب بدون عذر 2**: خصم من الراتب
- **غياب بدون عذر 3**: إنهاء التعاقد

#### سياسة التأخير / Late Policy

**للسائقين**:
- **تأخير 1**: تنبيه
- **تأخير متكرر (>10%)**: مراجعة أداء
- **تأخير مزمن**: برنامج تحسين إلزامي
- **عدم تحسن**: إعادة تعيين أو إنهاء

**للتنفيذ**:
- **تأخير استثنائي**: مقبول مع تفسير
- **تأخير متكرر**: تدريب إضافي
- **تأخير مزمن**: إعادة تقييم القدرة

### 3. التدريب المستمر / Continuous Training

#### برامج التدريب / Training Programs
- **التأهيل الأولي**: 2-3 أيام للجدد
- **التدريب الدوري**: ربع سنوي لجميع العاملين
- **تدريب متقدم**: لأفضل الأداء
- **تدريب إصلاحي**: للأداء الضعيف

#### المواضيع / Topics
- السلامة والصحة
- معايير الجودة
- استخدام الأنظمة والتطبيقات
- خدمة العملاء
- الامتثال واللوائح

### 4. التقييم والتطوير / Evaluation and Development

#### التقييم الدوري / Periodic Evaluation
- **أسبوعي**: مراجعة سريعة للمقاييس
- **شهري**: جلسة تغذية راجعة فردية
- **ربع سنوي**: تقييم شامل للأداء
- **سنوي**: مراجعة شاملة وخطة تطوير

#### مسارات التطوير / Development Paths
- من عامل إلى قائد فريق
- من سائق إلى مدرب سائقين
- من قائد فريق إلى مدير عمليات
- تخصصات (جودة، تدريب، تحسين)
