# 12) الاقتصاديات / Economics

## اقتصاديات الوحدة / Unit Economics

### نموذج الربح لكل طلب / Profit per Order Model

```
الربح/الطلب = الإيرادات − (تكلفة الميل الأخير + تكلفة الخدمة + الخسارة/الاحتيال + تكلفة الدعم)

Profit/Order = Revenue − (Last-Mile Cost + Service Cost + Loss/Fraud + Support Cost)
```

### تفصيل المكونات / Component Breakdown

#### 1. الإيرادات / Revenue
- **عمولة الطلب**: 10-20% من قيمة الطلب
- **رسوم التوصيل**: $1-5 (حسب المسافة والوقت)
- **رسوم خدمة**: 2-5% من قيمة الطلب
- **إيرادات إضافية**: اشتراكات، إعلانات

**مثال**:
```
قيمة الطلب: $50
عمولة (15%): $7.50
رسوم توصيل: $3.00
رسوم خدمة (3%): $1.50
إجمالي الإيرادات: $12.00
```

#### 2. تكلفة الميل الأخير / Last-Mile Cost
- **تعويض السائق**: $1.50-3.00 لكل توصيل
- **وقود/صيانة**: $0.30-0.50 لكل توصيل
- **إدارة الأسطول**: $0.20-0.30 لكل توصيل

**مثال**:
```
تعويض سائق: $2.00
وقود: $0.40
إدارة: $0.25
إجمالي الميل الأخير: $2.65
```

#### 3. تكلفة الخدمة / Service Cost
- **معالجة الطلب**: $0.20-0.40
- **الانتقاء والتعبئة**: $0.50-1.00
- **معالجة الدفع**: 2-3% من قيمة الطلب
- **بنية تحتية وتقنية**: $0.30-0.50

**مثال**:
```
معالجة: $0.30
انتقاء وتعبئة: $0.75
دفع (2.5%): $1.25
تقنية: $0.40
إجمالي الخدمة: $2.70
```

#### 4. الخسارة والاحتيال / Loss and Fraud
- **منتجات تالفة**: 1-2% من قيمة الطلب
- **احتيال العملاء**: 0.5-1% من قيمة الطلب
- **أخطاء التسليم**: 0.5-1% من قيمة الطلب

**مثال**:
```
تلف: $0.75
احتيال: $0.50
أخطاء: $0.50
إجمالي الخسارة: $1.75
```

#### 5. تكلفة الدعم / Support Cost
- **دعم العملاء**: $0.20-0.50 لكل طلب (متوسط)
- **معالجة الإرجاع**: $0.10-0.30 لكل طلب

**مثال**:
```
دعم: $0.30
إرجاع: $0.15
إجمالي الدعم: $0.45
```

### حساب الربح / Profit Calculation

```
الإيرادات: $12.00
− الميل الأخير: $2.65
− الخدمة: $2.70
− الخسارة: $1.75
− الدعم: $0.45
= هامش المساهمة المباشرة: $4.45
= النسبة: 37% من الإيرادات أو 8.9% من قيمة الطلب
```

### التباين حسب الفئة / Variation by Category

| الفئة / Category | متوسط قيمة الطلب | عمولة | هامش | ملاحظات |
|-----------------|------------------|-------|------|---------|
| منتجات جافة / Dry | $30 | 15% | 35% | تكاليف منخفضة |
| طازج / Fresh | $40 | 18% | 25% | تعامل خاص، فاقد أعلى |
| مبرد / Refrigerated | $50 | 20% | 30% | سلسلة تبريد |
| مجمد / Frozen | $60 | 22% | 28% | معدات خاصة |

## تجارب التسعير / Pricing Experiments

### 1. الرسوم الديناميكية / Dynamic Fees

#### العوامل المؤثرة / Influencing Factors
- **الوقت**: رسوم أعلى في أوقات الذروة
- **المسافة**: رسوم تتناسب مع المسافة
- **الطلب**: رسوم surge عند ارتفاع الطلب
- **العرض**: خصومات عند وفرة السائقين

#### مثال تطبيق / Example Application
```
رسوم أساسية: $2.00
+ ذروة (×1.5): $1.00
+ مسافة طويلة: $1.00
= إجمالي: $4.00
```

#### الاختبار / Testing
- **A/B Testing**: مقارنة نماذج مختلفة
- **تدريجي**: تطبيق على نسبة صغيرة أولاً
- **المراقبة**: تأثير على الطلب والإيرادات

### 2. الحد الأدنى للطلب / Minimum Order

#### الأسباب / Rationale
- ضمان ربحية كل توصيل
- تشجيع سلات أكبر
- تغطية التكاليف الثابتة

#### الاستراتيجيات / Strategies
- **حد ثابت**: $15-25 لكل المناطق
- **حد متغير**: حسب المنطقة والوقت
- **استثناءات**: للمشتركين أو العملاء المخلصين
- **رسوم صغيرة**: $2-3 للطلبات الأقل من الحد

#### التأثير / Impact
```
قبل الحد الأدنى:
- متوسط قيمة الطلب: $20
- عدد الطلبات: 1000
- إيرادات: $20,000

بعد الحد الأدنى $25:
- متوسط قيمة الطلب: $30
- عدد الطلبات: 850
- إيرادات: $25,500 (+27.5%)
```

### 3. رسوم الفئات الحساسة / Sensitive Category Fees

#### الفئات الحساسة / Sensitive Categories
- **سلسلة التبريد**: +$1-2 للمبرد/المجمد
- **القابل للكسر**: +$0.50-1 للزجاج/الخزف
- **عالي القيمة**: +1-2% للمنتجات الغالية
- **كبير الحجم**: +$1-3 للمنتجات الكبيرة

#### التبرير / Justification
- تكاليف معدات خاصة
- تدريب إضافي للعاملين
- مخاطر أعلى
- عناية أكثر في التعامل

#### الشفافية / Transparency
- إظهار واضح للرسوم الإضافية
- شرح السبب للعميل
- خيارات بديلة إن وجدت

## حدود القرار / Decision Boundaries

### عدم التوسع للفئات ذات الخسارة المستمرة / No Expansion for Persistent Loss Categories

#### المعايير / Criteria
- **خسارة لـ 3 أشهر متتالية**: هامش سلبي مستمر
- **لا مسار للربحية**: بعد محاولات التحسين
- **حجم صغير**: لا يبرر الاستثمار المستمر

#### عملية القرار / Decision Process

1. **الشهر 1 - اكتشاف الخسارة / Month 1 - Loss Discovery**
   - تحليل اقتصاديات الوحدة
   - تحديد أسباب الخسارة
   - وضع خطة تحسين

2. **الشهر 2 - تطبيق التحسينات / Month 2 - Improvements**
   - تجارب تسعير
   - تحسين العمليات
   - تقليل الفاقد
   - شراكات أفضل

3. **الشهر 3 - التقييم / Month 3 - Evaluation**
   - قياس التحسن
   - إذا لا زال الهامش سلبي:
     - **القرار**: إيقاف الفئة أو تقليل جذري

#### الاستثناءات / Exceptions
- **فئات استراتيجية**: ضرورية لاكتمال العرض
- **فئات نمو**: خسارة مؤقتة مع نمو واعد
- **فئات مجمعة**: ربحية إجمالية إيجابية

### الحد الأدنى لاقتصاديات الوحدة / Minimum Unit Economics

#### المتطلبات / Requirements
- **هامش مساهمة موجب**: >0% لكل طلب
- **LTV/CAC > 3**: القيمة الدائمة > 3× تكلفة الاكتساب
- **نقطة التعادل**: <12 شهر من إطلاق السوق

#### الإجراءات عند عدم التحقق / Actions if Not Met
1. **تحليل جذري**: لماذا لا نحقق الأهداف؟
2. **تجارب سريعة**: لتحسين الاقتصاديات
3. **قرار في 90 يوم**: استمرار أو إيقاف

## النمذجة المالية / Financial Modeling

### نموذج الطلب الواحد / Single Order Model

```
Excel/Sheets template:

| عنصر | قيمة | نسبة |
|------|------|------|
| قيمة الطلب | $50.00 | 100% |
| الإيرادات | $12.00 | 24% |
```

## مؤشرات الأداء المالية / Financial KPIs

### مؤشرات الصحة / Health Metrics
- **هامش المساهمة / Contribution Margin**: >8%
- **نقطة التعادل / Breakeven**: <12 شهر
- **نمو الإيرادات / Revenue Growth**: >20% شهرياً (مرحلة النمو)
- **معدل الحرق / Burn Rate**: مسار واضح للربحية

### مؤشرات الكفاءة / Efficiency Metrics
- **تكلفة الميل الأخير / Last-Mile Cost**: <$3 لكل طلب
- **تكلفة اكتساب العميل / CAC**: <$20
- **القيمة الدائمة / LTV**: >$60
- **نسبة LTV/CAC**: >3

### مؤشرات النمو / Growth Metrics
- **الطلبات الشهرية / Monthly Orders**: نمو ثابت
- **العملاء النشطون / Active Customers**: نمو ثابت
- **معدل التكرار / Repeat Rate**: >50%
- **متوسط قيمة الطلب / AOV**: نمو تدريجي
