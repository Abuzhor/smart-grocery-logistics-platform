# 6) المخاطر والضوابط / Risks and Controls

    - العائلة/ولي الأمر
    - ضوابط العائلة: صلاحيات، موافقات، حدود إنفاق، تنبيهات.
    - دعم قرار ولي الأمر: ميزانيات، بدائل، قوائم احتياج، توصيات.
    - الشركاء (تجار/مزارع/مصانع/مستودعات/طرف لوجستي ثالث)
        - إجراءات تشغيل معيارية + قياس أداء + تسعير مرن حسب الجودة.
        - تكاملات (تخطيط موارد المنشأة/إدارة المستودعات/نقاط البيع) حسب حجم الشريك.
    - القوى العاملة
        - حوافز مرتبطة بالجودة + تحسين المسارات + تقدير وقت الوصول + تجميع المهام.
    - الجهات التنظيمية والحكومية
        - حزمة الجهات: تقارير امتثال + بيانات مجمّعة + تكاملات برمجية (للقراءة فقط أولاً).
    - المنافسون
        - سوق عادل: قواعد حيادية تمنع الإقصاء وتفتح باب التكامل.
