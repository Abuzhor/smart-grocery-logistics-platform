# 13) قاموس القياس / Measurement Dictionary

## مقاييس العمليات / Operations Metrics

### التسليم / Delivery

#### On-Time Delivery (OTD) - التسليم في الوقت المحدد
- **التعريف**: نسبة الطلبات المسلمة خلال نافذة الوقت الموعودة
- **الحساب**: (عدد الطلبات في الوقت / إجمالي الطلبات) × 100
- **الهدف**: ≥95%
- **التكرار**: يومي، أسبوعي، شهري
- **المصدر**: نظام إدارة التوصيل

#### Delivery Cost per Order - تكلفة التوصيل لكل طلب
- **التعريف**: متوسط تكلفة الميل الأخير لكل طلب
- **الحساب**: إجمالي تكاليف التوصيل / عدد الطلبات
- **الهدف**: $2-4 (حسب المدينة)
- **التكرار**: يومي
- **المصدر**: نظام المحاسبة + نظام التوصيل

#### First Attempt Delivery Rate - معدل التسليم من أول محاولة
- **التعريف**: نسبة الطلبات المسلمة من المحاولة الأولى
- **الحساب**: (طلبات المحاولة الأولى / إجمالي الطلبات) × 100
- **الهدف**: ≥95%
- **التكرار**: يومي
- **المصدر**: نظام إدارة التوصيل

### التنفيذ / Fulfillment

#### Picking Accuracy - دقة الانتقاء
- **التعريف**: نسبة المنتجات المنتقاة بشكل صحيح
- **الحساب**: (منتجات صحيحة / إجمالي المنتجات المنتقاة) × 100
- **الهدف**: ≥98%
- **التكرار**: يومي
- **المصدر**: نظام إدارة المستودعات

#### Picking Time - وقت الانتقاء
- **التعريف**: متوسط الوقت لانتقاء طلب واحد
- **الحساب**: إجمالي وقت الانتقاء / عدد الطلبات
- **الهدف**: <15 دقيقة لطلب متوسط
- **التكرار**: يومي
- **المصدر**: نظام إدارة المستودعات

#### Substitution Rate - معدل الاستبدال
- **التعريف**: نسبة المنتجات المستبدلة من إجمالي المنتجات
- **الحساب**: (منتجات مستبدلة / إجمالي المنتجات) × 100
- **الهدف**: <5%
- **التكرار**: يومي
- **المصدر**: نظام إدارة الطلبات

#### Substitution Acceptance - قبول الاستبدال
- **التعريف**: نسبة الاستبدالات المقبولة من العملاء
- **الحساب**: (استبدالات مقبولة / إجمالي الاستبدالات) × 100
- **الهدف**: ≥80%
- **التكرار**: أسبوعي
- **المصدر**: نظام إدارة الطلبات

### المخزون / Inventory

#### Inventory Accuracy - دقة المخزون
- **التعريف**: نسبة تطابق المخزون الفعلي مع المخزون في النظام
- **الحساب**: (عناصر متطابقة / إجمالي العناصر المدققة) × 100
- **الهدف**: ≥95%
- **التكرار**: أسبوعي
- **المصدر**: نظام إدارة المخزون

#### Stockout Rate - معدل نفاد المخزون
- **التعريف**: نسبة المنتجات غير المتوفرة عند الطلب
- **الحساب**: (طلبات فيها منتجات غير متوفرة / إجمالي الطلبات) × 100
- **الهدف**: <5%
- **التكرار**: يومي
- **المصدر**: نظام إدارة المخزون

#### Waste Rate - معدل الهدر
- **التعريف**: نسبة المنتجات التالفة أو منتهية الصلاحية
- **الحساب**: (قيمة المنتجات المهدرة / إجمالي قيمة المخزون) × 100
- **الهدف**: <2% للطازج، <0.5% للجاف
- **التكرار**: أسبوعي
- **المصدر**: نظام إدارة المخزون

## مقاييس العملاء / Customer Metrics

### الرضا / Satisfaction

#### Customer Satisfaction Index (CSI) - مؤشر رضا العملاء
- **التعريف**: مؤشر مركب لرضا العملاء
- **الحساب**: متوسط التقييمات (1-5) × 20
- **الهدف**: ≥60/100 (أي ≥3.0/5.0)
- **التكرار**: يومي
- **المصدر**: نظام التقييمات

#### Net Promoter Score (NPS) - صافي نقاط الترويج
- **التعريف**: احتمالية توصية العملاء بالخدمة
- **الحساب**: (% المروجين) - (% المنتقدين)
- **الهدف**: >30
- **التكرار**: شهري
- **المصدر**: استبيانات العملاء

### السلوك / Behavior

#### Repeat Rate - معدل التكرار
- **التعريف**: نسبة العملاء الذين يطلبون مرة أخرى خلال 30 يوم
- **الحساب**: (عملاء مكررين / إجمالي العملاء الجدد) × 100
- **الهدف**: ≥50%
- **التكرار**: شهري
- **المصدر**: نظام إدارة العملاء

#### Order Frequency - تكرار الطلبات
- **التعريف**: متوسط عدد الطلبات لكل عميل نشط شهرياً
- **الحساب**: إجمالي الطلبات / عدد العملاء النشطين
- **الهدف**: >2 طلبات/شهر
- **التكرار**: شهري
- **المصدر**: نظام إدارة الطلبات

#### Cancellation Rate - معدل الإلغاء
- **التعريف**: نسبة الطلبات الملغاة من إجمالي الطلبات
- **الحساب**: (طلبات ملغاة / إجمالي الطلبات) × 100
- **الهدف**: ≤5%
- **التكرار**: يومي
- **المصدر**: نظام إدارة الطلبات

### القيمة / Value

#### Average Order Value (AOV) - متوسط قيمة الطلب
- **التعريف**: متوسط قيمة كل طلب
- **الحساب**: إجمالي قيمة الطلبات / عدد الطلبات
- **الهدف**: $40-60 (حسب السوق)
- **التكرار**: يومي
- **المصدر**: نظام إدارة الطلبات

#### Customer Lifetime Value (LTV) - القيمة الدائمة للعميل
- **التعريف**: القيمة المتوقعة للعميل طوال فترة العلاقة
- **الحساب**: (AOV × Order Frequency × Average Lifespan) − CAC
- **الهدف**: >$60
- **التكرار**: ربع سنوي
- **المصدر**: نظام التحليلات

## مقاييس الشركاء / Partner Metrics

### الجودة / Quality

#### Partner Quality Score - درجة جودة الشريك
- **التعريف**: مؤشر مركب لأداء الشريك
- **الحساب**: متوسط موزون (دقة مخزون 25% + سرعة تنفيذ 20% + جودة منتجات 25% + رضا عملاء 15% + امتثال 15%)
- **الهدف**: ≥80/100
- **التكرار**: أسبوعي
- **المصدر**: نظام إدارة الشركاء

#### Product Quality Rate - معدل جودة المنتجات
- **التعريف**: نسبة المنتجات المقبولة من الشريك
- **الحساب**: (منتجات مقبولة / إجمالي المنتجات) × 100
- **الهدف**: ≥98%
- **التكرار**: أسبوعي
- **المصدر**: نظام الجودة

### الأداء / Performance

#### Fulfillment Speed - سرعة التنفيذ
- **التعريف**: متوسط الوقت من استلام الطلب إلى جاهزيته للتوصيل
- **الحساب**: إجمالي وقت التنفيذ / عدد الطلبات
- **الهدف**: <30 دقيقة
- **التكرار**: يومي
- **المصدر**: نظام إدارة التنفيذ

#### Partner Uptime - وقت تشغيل الشريك
- **التعريف**: نسبة الوقت الذي يكون فيه الشريك متاح للطلبات
- **الحساب**: (ساعات التشغيل الفعلية / ساعات التشغيل المخططة) × 100
- **الهدف**: ≥95%
- **التكرار**: أسبوعي
- **المصدر**: نظام إدارة الشركاء

## المقاييس المالية / Financial Metrics

### الإيرادات / Revenue

#### Gross Merchandise Value (GMV) - إجمالي قيمة البضائع
- **التعريف**: إجمالي قيمة جميع الطلبات قبل الخصومات
- **الحساب**: ∑ قيمة كل طلب
- **الهدف**: نمو شهري >15%
- **التكرار**: يومي
- **المصدر**: نظام إدارة الطلبات

#### Revenue - الإيرادات
- **التعريف**: إجمالي الإيرادات المحققة (عمولات + رسوم)
- **الحساب**: عمولات + رسوم توصيل + رسوم خدمة + اشتراكات + إعلانات
- **الهدف**: حسب الخطة
- **التكرار**: يومي
- **المصدر**: نظام المحاسبة

#### Take Rate - معدل الاستقطاع
- **التعريف**: نسبة الإيرادات من GMV
- **الحساب**: (الإيرادات / GMV) × 100
- **الهدف**: 20-30%
- **التكرار**: شهري
- **المصدر**: نظام التحليلات المالية

### الربحية / Profitability

#### Contribution Margin - هامش المساهمة
- **التعريف**: الإيرادات ناقص التكاليف المتغيرة
- **الحساب**: الإيرادات − (تكلفة الميل الأخير + الخدمة + الخسارة + الدعم)
- **الهدف**: موجب وينمو
- **التكرار**: يومي
- **المصدر**: نظام التحليلات المالية

#### Contribution Margin % - نسبة هامش المساهمة
- **التعريف**: هامش المساهمة كنسبة من الإيرادات
- **الحساب**: (هامش المساهمة / الإيرادات) × 100
- **الهدف**: ≥30%
- **التكرار**: شهري
- **المصدر**: نظام التحليلات المالية

### الكفاءة / Efficiency

#### Customer Acquisition Cost (CAC) - تكلفة اكتساب العميل
- **التعريف**: متوسط تكلفة اكتساب عميل جديد
- **الحساب**: إجمالي مصاريف التسويق والمبيعات / عدد العملاء الجدد
- **الهدف**: <$20
- **التكرار**: شهري
- **المصدر**: نظام التسويق + المحاسبة

#### LTV/CAC Ratio - نسبة LTV إلى CAC
- **التعريف**: نسبة القيمة الدائمة للعميل إلى تكلفة اكتساب العميل
- **الحساب**: LTV / CAC
- **الهدف**: >3
- **التكرار**: ربع سنوي
- **المصدر**: نظام التحليلات

## مقاییس الجودة والامتثال / Quality and Compliance Metrics

### الجودة / Quality

#### Order Accuracy - دقة الطلب
- **التعريف**: نسبة الطلبات الكاملة والصحيحة
- **الحساب**: (طلبات صحيحة تماماً / إجمالي الطلبات) × 100
- **الهدف**: ≥95%
- **التكرار**: يومي
- **المصدر**: نظام إدارة الطلبات

#### Complaint Rate - معدل الشكاوى
- **التعريف**: نسبة الطلبات التي تلقت شكوى
- **الحساب**: (طلبات بها شكوى / إجمالي الطلبات) × 100
- **الهدف**: <3%
- **التكرار**: يومي
- **المصدر**: نظام دعم العملاء

### الامتثال / Compliance

#### Audit Pass Rate - معدل نجاح التدقيق
- **التعريف**: نسبة عمليات التدقيق الناجحة
- **الحساب**: (عمليات تدقيق ناجحة / إجمالي عمليات التدقيق) × 100
- **الهدف**: ≥95%
- **التكرار**: ربع سنوي
- **المصدر**: نظام إدارة الامتثال

#### Fraud Rate - معدل الاحتيال
- **التعريف**: نسبة المعاملات المشبوهة أو الاحتيالية
- **الحساب**: (معاملات احتيالية مؤكدة / إجمالي المعاملات) × 100
- **الهدف**: <2%
- **التكرار**: أسبوعي
- **المصدر**: نظام مكافحة الاحتيال

## مقاييس النظام / System Metrics

### الأداء / Performance

#### System Uptime - وقت تشغيل النظام
- **التعريف**: نسبة الوقت الذي يكون فيه النظام متاحاً
- **الحساب**: (وقت التشغيل / إجمالي الوقت) × 100
- **الهدف**: ≥99.9%
- **التكرار**: يومي
- **المصدر**: نظام المراقبة

#### Response Time (p95) - زمن الاستجابة
- **التعريف**: الزمن الذي يستغرقه النظام للاستجابة للطلبات (percentile 95)
- **الحساب**: الزمن الذي تستجيب فيه 95% من الطلبات
- **الهدف**: <200ms
- **التكرار**: في الوقت الفعلي
- **المصدر**: نظام المراقبة

#### Error Rate - معدل الأخطاء
- **التعريف**: نسبة الطلبات التي تنتج أخطاء
- **الحساب**: (طلبات بها أخطاء / إجمالي الطلبات) × 100
- **الهدف**: <0.1%
- **التكرار**: في الوقت الفعلي
- **المصدر**: نظام المراقبة
