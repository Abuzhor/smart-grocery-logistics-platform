# 7) المقاييس وبوابات القرار / Metrics and Gates

## المقاييس النجمية الشمالية / North-Star Metrics

### 1. الامتثال في الوقت المحدد / On-Time Compliance
- **التعريف**: نسبة الطلبات المسلمة في نافذة الوقت الموعودة
- **الهدف**: ≥95%
- **القياس**: يومي، أسبوعي، شهري
- **الأهمية**: مقياس أساسي لرضا العملاء والموثوقية

### 2. نجاح الدفع / Payment Success
- **التعريف**: نسبة المعاملات المالية الناجحة من إجمالي المحاولات
- **الهدف**: ≥99%
- **القياس**: في الوقت الفعلي
- **الأهمية**: حرج للإيرادات وتجربة العملاء

### 3. معدل الإلغاء / Cancellation Rate
- **التعريف**: نسبة الطلبات الملغاة (من العميل أو النظام) من إجمالي الطلبات
- **الهدف**: ≤5%
- **القياس**: يومي، بتقسيم حسب سبب الإلغاء
- **الأهمية**: مؤشر على رضا العملاء وجودة الخدمة

### 4. جودة الشريك / Partner Quality
- **التعريف**: مؤشر مركب من دقة المخزون، سرعة التنفيذ، جودة المنتجات
- **الهدف**: درجة ≥80/100
- **القياس**: أسبوعي لكل شريك
- **الأهمية**: ضمان جودة متسقة عبر الشبكة

### 5. تكلفة الميل الأخير / Last-Mile Cost
- **التعريف**: متوسط تكلفة التوصيل لكل طلب
- **الهدف**: $2-4 (يختلف حسب المدينة والفئة)
- **القياس**: يومي، مع تحليل اتجاهات
- **الأهمية**: حرج لاقتصاديات الوحدة

### 6. هامش المساهمة المباشرة لكل فئة / Direct Contribution Margin per Category
- **التعريف**: (الإيرادات - التكاليف المباشرة) / الإيرادات لكل فئة منتج
- **الهدف**: موجب لجميع الفئات الأساسية
- **القياس**: شهري
- **الأهمية**: قرارات التوسع واستمرار الفئات

## بوابات التوسع / Expansion Gates

### شروط GO / GO Conditions

لإطلاق مدينة جديدة أو فئة منتج جديدة، يجب تحقيق:

#### 1. الموثوقية / Reliability
- **KPIs مستقرة**: جميع المقاييس النجمية ضمن الأهداف لمدة شهرين متتاليين
- **تقليل الحوادث**: تخفيض بنسبة 50% في الحوادث الحرجة مقارنة بالربع السابق
- **وقت تشغيل النظام**: ≥99.5% uptime

#### 2. الاقتصاديات / Economics
- **CAC مقابل LTV**: نسبة LTV/CAC > 3
- **اقتصاديات الوحدة الإيجابية**: هامش مساهمة موجب لـ 80% من الطلبات
- **نقطة التعادل**: الوصول لنقطة التعادل التشغيلي في السوق الحالي

#### 3. الثقة / Trust
- **النزاعات والاحتيال**: <2% من إجمالي الطلبات
- **تقييمات العملاء**: متوسط ≥4.0/5.0
- **معدل الاحتفاظ**: ≥60% من العملاء يكررون الطلب خلال 30 يوم

#### 4. الامتثال / Compliance
- **سياسات موثقة**: جميع سياسات التشغيل والامتثال موثقة ومراجعة
- **إطار تدقيق**: نظام audit trail كامل وعامل
- **استعداد تنظيمي**: الحصول على جميع التراخيص والموافقات المطلوبة

### شروط STOP / STOP Conditions

يجب إيقاف التوسع إذا:

- **خسائر مستمرة**: هامش سلبي لـ 3 أشهر متتالية في أي فئة
- **مشاكل جودة**: معدل شكاوى >10% أو حوادث صحية/سلامة
- **عدم استقرار النظام**: >3 حوادث حرجة في شهر واحد
- **مشاكل امتثال**: انتهاكات تنظيمية أو قانونية
- **مخاطر احتيال**: احتيال >5% من المعاملات

## الأهداف والنتائج الرئيسية الفصلية / Quarterly OKRs

### مثال Q1 2026

#### الهدف 1: التميز التشغيلي / Operational Excellence
**النتائج الرئيسية**:
- التسليم في الوقت المحدد ≥95%
- معدل الإلغاء ≤5%
- جودة الشريك ≥80/100

#### الهدف 2: الاستدامة المالية / Financial Sustainability
**النتائج الرئيسية**:
- نجاح الدفع ≥99%
- هامش مساهمة مباشر موجب لكل فئة أساسية
- تكلفة الميل الأخير <$3.5

#### الهدف 3: رضا العملاء / Customer Satisfaction
**النتائج الرئيسية**:
- مؤشر رضا العملاء ≥60/100
- معدل تكرار الطلب ≥50%
- معدل الشكاوى <3%

#### الهدف 4: الامتثال والأمان / Compliance and Security
**النتائج الرئيسية**:
- صفر انتهاكات امتثال مادية
- معدل احتيال <2%
- صفر اختراقات بيانات

## مؤشرات الأداء التشغيلية / Operational KPIs

### كفاءة التنفيذ / Fulfillment Efficiency
- **وقت الانتقاء**: متوسط الوقت لانتقاء طلب
- **دقة الانتقاء**: نسبة المنتجات الصحيحة
- **معدل الاستبدال**: نسبة المنتجات المستبدلة

### كفاءة التوصيل / Delivery Efficiency
- **أوامر لكل ساعة**: عدد التوصيلات لكل سائق بالساعة
- **الأميال لكل توصيل**: متوسط المسافة لكل طلب
- **معدل التوصيل من أول محاولة**: نسبة التسليم الناجح من أول مرة

### الجودة / Quality
- **دقة الطلب**: نسبة الطلبات بدون أخطاء
- **حوادث سلسلة التبريد**: عدد حوادث انقطاع التبريد
- **معدل التلف**: نسبة المنتجات التالفة

### الصحة المالية / Financial Health
- **متوسط قيمة الطلب (AOV)**: Average Order Value
- **تكلفة اكتساب العميل (CAC)**: Customer Acquisition Cost
- **القيمة الدائمة (LTV)**: Customer Lifetime Value
- **الإيرادات لكل مستخدم نشط**: ARPU

## لوحات القيادة / Dashboards

### لوحة القيادة اليومية / Daily Dashboard
- حالة المقاييس النجمية الحالية
- تنبيهات للحوادث والاستثناءات
- أداء المدن والشركاء الأساسيين

### لوحة القيادة الأسبوعية / Weekly Dashboard
- اتجاهات المقاييس الأسبوعية
- تحليل الانحرافات
- تقدم OKRs

### لوحة القيادة الشهرية / Monthly Dashboard
- مراجعة شاملة للأداء
- تحليل اقتصاديات الوحدة
- مراجعة المخاطر والامتثال
- توصيات للتحسين
