# 5) نموذج الإيرادات / Revenue Model

## مصادر الإيرادات / Revenue Streams

### 1. العمولات / Commission

#### عمولة على الطلبات / Order Commission
- **النسبة**: 10-20% من قيمة الطلب (حسب الفئة والشريك)
- **التطبيق**: على جميع الطلبات المكتملة
- **التمييز**: نسب مختلفة حسب:
  - فئة المنتج (طازج، مجمد، جاف)
  - حجم الشريك والأداء
  - حجم الطلب

#### عمولة التسليم / Delivery Fee Commission
- **النسبة**: 15-25% من رسوم التوصيل
- **العوامل**: المسافة، الوقت، الطلب على الخدمة
- **الديناميكية**: تسعير ديناميكي في أوقات الذروة

### 2. الاشتراكات / Subscriptions

#### اشتراكات الشركاء / Partner Subscriptions
- **باقة أساسية (Basic)**: 200-500 دولار شهرياً
  - وصول محدود للمنصة
  - أدوات أساسية لإدارة الطلبات
  - تقارير شهرية
  
- **باقة احترافية (Professional)**: 500-1000 دولار شهرياً
  - جميع ميزات الباقة الأساسية
  - تحليلات متقدمة
  - أولوية في الظهور
  - دعم مخصص
  
- **باقة مؤسسات (Enterprise)**: مخصصة
  - حلول مخصصة
  - تكامل API متقدم
  - مدير حساب مخصص

#### اشتراكات العائلات / Family Subscriptions
- **باقة بريميوم للعملاء (Customer Premium)**: 10-30 دولار شهرياً
  - توصيل مجاني أو مخفض
  - وصول مبكر للعروض
  - دعم أولوية
  - خصومات حصرية

### 3. الإعلانات / Advertising

#### إعلانات على المنصة / On-Platform Advertising
- **منتجات مُروّجة (Sponsored Products)**: ظهور المنتجات في نتائج البحث الأولى
- **علامات تجارية مُروّجة (Sponsored Brands)**: عرض العلامات التجارية بشكل بارز
- **إعلانات العرض (Display Ads)**: لافتات على الصفحة الرئيسية والفئات
- **التسعير**: تكلفة النقرة (CPC) أو تكلفة الظهور (CPM)

### 4. خدمات B2B / B2B Services

#### التوزيع بالجملة / Wholesale Distribution
- **الهامش**: 5-10% على طلبات الجملة
- **العملاء**: المطاعم، المقاهي، الفنادق، المكاتب
- **القيمة المضافة**: تسليم منتظم، فوترة موحدة، تقارير

#### الخدمات اللوجستية / Logistics Services
- **رسوم الخدمة**: استخدام شبكة التوصيل لأطراف ثالثة
- **التطبيقات**: توصيل طرود، نقل بضائع

### 5. خدمات الامتثال والتقارير / Compliance and Reporting Services

#### تقارير الامتثال / Compliance Reports
- **للجهات التنظيمية**: تقارير مفصلة للامتثال
- **للشركاء**: تقارير جودة وأداء قابلة للتدقيق
- **التسعير**: رسوم سنوية أو حسب التقرير

#### الاستشارات / Consulting
- **خدمات**: تحسين العمليات، التدريب، إعداد السياسات
- **التسعير**: بالساعة أو بالمشروع

## نموذج التسعير / Pricing Model

### التسعير الديناميكي / Dynamic Pricing
- تعديل الرسوم حسب:
  - العرض والطلب
  - أوقات الذروة
  - الموسمية
  - المنطقة الجغرافية

### الحد الأدنى للطلب / Minimum Order
- حد أدنى لقيمة الطلب لضمان الربحية
- يختلف حسب المنطقة وتكاليف التوصيل

### رسوم الفئات الحساسة / Sensitive Category Fees
- رسوم إضافية للمنتجات:
  - المبردة والمجمدة (سلسلة التبريد)
  - القابلة للكسر
  - عالية القيمة

## التوقعات المالية / Financial Projections

### المرحلة الأولى (السنة 1) / Phase 1 (Year 1)
- التركيز على اكتساب السوق
- الإيرادات الأساسية من العمولات
- هوامش ربح منخفضة (5-10%)

### مرحلة النمو (السنوات 2-3) / Growth Phase (Years 2-3)
- زيادة الإيرادات من الاشتراكات والإعلانات
- تحسين الهوامش (15-25%)
- توسع جغرافي

### مرحلة النضج (السنة 4+) / Maturity Phase (Year 4+)
- تنويع كامل لمصادر الإيرادات
- هوامش ربح مستقرة (25-35%)
- إيرادات B2B كبيرة

## مؤشرات الأداء المالية / Financial KPIs

- **متوسط قيمة الطلب (AOV)**: Average Order Value
- **الإيرادات لكل مستخدم (ARPU)**: Average Revenue Per User
- **القيمة الدائمة للعميل (LTV)**: Customer Lifetime Value
- **تكلفة اكتساب العميل (CAC)**: Customer Acquisition Cost
- **نسبة LTV/CAC**: يجب أن تكون > 3
- **معدل الاحتفاظ بالعملاء**: Customer Retention Rate
- **هامش المساهمة المباشرة**: Direct Contribution Margin
