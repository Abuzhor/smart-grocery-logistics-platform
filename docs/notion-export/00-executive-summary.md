# 0) الملخص التنفيذي / Executive Summary

    - هذه المنصة هي **سوق وخدمات متعدد الأطراف** يتحول من مجرد “توصيل تموينات” إلى **طبقة تشغيل وبيانات** تربط: الإنتاج → التخزين → التوزيع → التوصيل → التنظيم.
    - الهدف العالمي: **إطلاق فئات جديدة ومدن ودول بسرعة** مع امتثال مضبوط وجودة قابلة للتدقيق واقتصاديات وحدة سليمة.
    - المبدأ التشغيلي: **كل فئة = تشغيل + امتثال + قياس**.
    - إطار عالمي مختصر (ملخص صفحة واحدة) - لرفع الكثافة والجاهزية الدولية
        - **المشكلة عالميًا**: تجزؤ العرض + اختلاف الامتثال + هشاشة آخر ميل + ضعف بيانات الجودة.
        - **الحل**: نواة منصة + حزم تشغيل للفئات + محرك سياسات للامتثال + طبقة قياس موحدة.
        - **تميّز قابل للتصدير**: إعدادات محلية دون تفريع منتج، مع نفس المقاييس وبوابات القرار.
        - **العميل/الطلب**: عائلات + أفراد + أعمال صغيرة + شركات (أعمال إلى أعمال) حسب المدينة.
        - **العرض**: تجزئة + سلاسل إمداد + خدمات ميدانية + طرف لوجستي ثالث.
        - **الميزة التنافسية**: جودة قابلة للتدقيق + اقتصاديات وحدة لكل فئة + بيانات أحداث.
        - **نموذج الإيراد**: (عمولة + توصيل) + اشتراكات شركاء/عائلات + إعلانات + أعمال إلى أعمال + خدمات امتثال/تقارير.
        - **مخاطر حرجة**: احتيال/نزاعات + امتثال متعدد الدول + جودة التبريد + دقة المخزون.
        - **إغلاق المخاطر**: التحقق من الهوية والتحقق من الشريك التجاري + إثبات التسليم قوي + إجراءات تصحيحية ووقائية + سجل تدقيق + سياسات احتفاظ بيانات.
        - **مقاييس شمالية موحدة**:
            - الالتزام بالوقت.
            - نجاح المدفوعات.
            - الإلغاء.
            - جودة الشريك.
            - تكلفة آخر ميل.
            - مساهمة ربحية مباشرة لكل فئة.
        - **بوابات التوسع (قرار الانطلاق أو الإيقاف)**:
            - الاعتمادية: مؤشرات أداء رئيسية + انخفاض الحوادث.
            - الاقتصاد: تكلفة اكتساب العميل مقابل القيمة على مدى حياة العميل + اقتصاديات وحدة موجبة.
            - الثقة: نزاعات/احتيال ضمن حد.
            - الامتثال: سياسات موثقة قابلة للتدقيق.
        - **خارطة 90 يومًا (إطلاق مدينة)**:
            - أسبوع 1–2: تصميم نطاقات التشغيل + تعريف المقاييس + سياسات الفئة.
            - أسبوع 3-6: الحد الأدنى من المنتج القابل للإطلاق (مستهلك/شريك/قوى عاملة) + إثبات التسليم + تسويات.
            - أسبوع 7-10: تشغيل تجريبي + تحسين تقدير وقت الوصول/المسارات + إجراءات تصحيحية ووقائية.
            - أسبوع 11–13: توسيع فئتين إضافيتين + تهيئة الدفع/الضرائب.
    - حزمة كثافة المنسدلات (دليل تشغيل جاهز) - لإخراج كل قسم كـ «حزمة تنفيذ + قياس»
        - قالب موحّد يُستخدم داخل كل منسدلة
            - **الهدف**: ماذا نحقق ولماذا.
            - **القرارات**: ما الذي يجب حسمه قبل البناء/الإطلاق.
            - **المخرجات**: وثائق/ميزات/عمليات ملموسة.
            - **التشغيل**: إجراءات تشغيل معيارية + تدريب + أدوار + سعات.
            - **الامتثال**: سياسة + متطلبات تحقق + سجل تدقيق.
            - **القياس**: مؤشرات أداء رئيسية + تعريفات + حدود قبول.
            - **المخاطر**: أهم 3 مخاطر + ضوابطها.
            - **بوابة قرار**: شروط قرار الانطلاق أو الإيقاف.
        - 1) الرؤية والأهداف — تكثيف تنفيذي
            - **الهدف الشمالي**: تكرار شراء مرتفع + موثوقية + اقتصاديات وحدة موجبة.
            - **الأهداف والنتائج الرئيسية مقترحة (ربع سنوي)**:
                - الالتزام بالوقت ≥ 95%.
                - الإلغاء ≤ 5%.
                - نجاح المدفوعات ≥ 99%.
                - مؤشر رضا العميل ≥ 60.
                - مساهمة ربحية مباشرة موجبة لكل فئة أساسية.
            - **قرار الانطلاق أو الإيقاف**: لا توسع قبل تحقق 4 أبعاد (اعتمادية/اقتصاد/ثقة/امتثال).
        - 2) السوق المستهدف — تكثيف تنفيذي
            - **معيار اختيار المدينة**: كثافة + وقت تنقل + جاهزية شركاء + تنظيم + تكلفة آخر ميل.
            - **مزيج البداية**: بقالة + خدمات منزلية بسيطة (لتوازن التكرار مع التعقيد).
            - **إستراتيجية دخول**: شراكات مرساة + تغطية نطاقات + عروض افتتاح + تجربة متسقة.
        - 3) نطاق المنصة — تكثيف تنفيذي
            - **تصنيف الفئات**: (غير حساسة/سلسلة تبريد/منظمة/خدمات مجدولة/أعمال إلى أعمال).
            - **حزم جاهزة**:
                - حزمة بقالة: بدائل + تجزئة طلب + هدف مستوى الخدمة تجهيز.
                - حزمة تبريد: حساسات/تحقق + حدود زمن.
                - حزمة منظمة: التحقق من الهوية والتحقق من الشريك التجاري + قيود + إثبات التسليم قوي.
                - حزمة خدمات: جدولة + تسعير خدمة + إثبات إنجاز.
        - 4) واجهات المنتج — تكثيف تنفيذي
            - **المستهلك**: رحلة موحّدة + دعم + نزاعات بالأدلة.
            - **الشريك**: كتالوج/مخزون/تجهيز/تسويات.
            - **القوى العاملة**: إسناد + تجميع + مسارات + إثبات التسليم.
            - **التشغيل**: مراقبة + جودة + حوادث + نزاعات.
        - 8) التقنية — تكثيف تنفيذي
            - **مبادئ معمارية**: منهجية مدفوعة بالنطاقات + منهجية مدفوعة بالأحداث + مفاتيح تفعيل الخصائص + متعدد المستأجرين جاهزية.
            - **الاعتمادية**: مؤشرات أداء رئيسية + الملاحظة والتشخيص + التعافي من الكوارث + إطلاق تدريجي.
            - **الأمن**: انعدام الثقة + تشفير + تصنيف بيانات + تدقيق.
        - 9) التشغيل — تكثيف تنفيذي
            - **إجراءات تشغيل معيارية إلزامية**: تجهيز/استبدال/إرجاع/استدعاء + إدارة حوادث + إجراءات تصحيحية ووقائية.
            - **حوكمة الشريك**: درجات جودة + تدقيق + خطة تحسين.
            - **حوكمة القوى العاملة**: حوافز جودة + سياسات إلغاء/تأخر.
        - 10) الاقتصاديات — تكثيف تنفيذي
            - **وحدة القياس**: ربح/طلب = (إيراد) − (آخر ميل + خدمة + خسائر/احتيال + دعم).
            - **تجارب تسعير**: رسوم ديناميكية + حد أدنى + رسوم فئات حساسة.
            - **حدود قرار**: لا توسيع لفئة بخسارة مباشرة مستمرة.
        - 11) القياس — تكثيف تنفيذي
            - **قاموس قياس**: تعريف كل مؤشر أداء رئيسي + طريقة حساب + مصدر بيانات.
            - **لوحات**:
                - لوحة يومية تشغيلية.
                - لوحة أسبوعية اقتصاديات وحدة.
                - لوحة شهرية صحة الشركاء.
        - 12) الإطلاق العالمي — تكثيف تنفيذي
            - **حزمة مدينة**: إعدادات محلية + دفع/ضرائب + نطاقات تشغيل + شركاء مرساة.
            - **حزمة دولة**: محرك سياسات + امتثال + بوابة تقارير.
        - 13) المخاطر — تكثيف تنفيذي
            - **أعلى عشرة مخاطر**: احتيال/نزاعات/تبريد/دقة مخزون/مدفوعات/امتثال/توافر/أمن/سمعة/اعتمادية.
            - **ضوابط**: التحقق من الهوية والتحقق من الشريك التجاري + إثبات التسليم + سجل تدقيق + إجراءات تصحيحية ووقائية + فصل صلاحيات.
        - تطبيق القالب على كل الأقسام (أدلة تشغيل لكل قسم)
            - 0) الملخص التنفيذي
                - **الهدف**: رسالة واحدة قابلة للعرض للمستثمر/الشريك/المنظّم.
                - **المخرجات**: ملخص صفحة واحدة + مخطط العرض التقديمي + قائمة الفرضيات.
                - **القياس**: وضوح العرض + اتساق المقاييس + جاهزية التوسع.
                - **بوابة قرار**: تعريف 3 حالات استخدام + 3 فئات إطلاق + 1 مدينة.
            - دليل تفصيلي (تشغيل + امتثال + قياس) لكل قسم
                - 0) الملخص التنفيذي — تفصيلي
                    - **القرارات**: هدف شمالي واحد + فئات الإطلاق + نموذج تشغيلي + نموذج إيراد + حدود المخاطر.
                    - **المخرجات**:
                        - وثيقة ملخص صفحة واحدة.
                        - وثيقة فرضيات (الفرضيات) مرتبة حسب المخاطر.
                        - خريطة مقاييس الهدف الشمالي + 5 مؤشرات أداء رئيسية.
                    - **التشغيل (إجراءات تشغيل معيارية)**:
                        - مراجعة أسبوعية للفرضيات.
                        - تقرير أسبوعي (جودة/اقتصاديات/مخاطر).
                    - **الامتثال**: تعريف نطاق البيانات + من يطلع عليها + ما الذي يُحذف ومتى.
                    - **القياس (مؤشرات أداء رئيسية)**: التكرار، الالتزام بالوقت، الإلغاء، نجاح الدفع، هامش/طلب.
                    - **بوابة قرار**: لا بناء دون تعريف مؤشر أداء رئيسي لكل فرضية + حد قبول.
                - 1) الرؤية والأهداف — تفصيلي
                    - **القرارات**: الأهداف والنتائج الرئيسية ربع سنوية + أهداف مستوى الخدمة + حدود الاقتصاديات.
                    - **المخرجات**: لوحة الأهداف والنتائج الرئيسية + تعريفات المقاييس + سياسة قرار الانطلاق أو الإيقاف.
                    - **التشغيل**: اجتماع أسبوعي مراجعة الأهداف والنتائج الرئيسية + اجتماع شهري مراجعة بوابات التوسع.
                    - **الامتثال**: مراجعة أثر الخصائص على الخصوصية قبل الإطلاق.
                    - **القياس**: معدل تحقيق الأهداف للـ الأهداف والنتائج الرئيسية + انحرافات أهداف مستوى الخدمة.
                    - **بوابة قرار**: لا توسع إذا فشل مؤشر أداء رئيسي محوري أسبوعين متتاليين.
                - 2) السوق المستهدف — تفصيلي
                    - **القرارات**: اختيار مدينة/منطقة + اختيار شريحة البداية + قنوات اكتساب.
                    - **المخرجات**:
                        - تقدير حجم السوق (إجمالي السوق القابل للخدمة والسوق المتاح والسوق المستهدف) مبسط.
                        - مشهد المنافسة + بدائل المستهلك.
                        - خطة دخول 90 يومًا.
                    - **التشغيل**: اختبار رسائل (ألف/باء) + تتبع القنوات.
                    - **الامتثال**: التحقق من القيود الإعلانية/الترويجية حسب الدولة.
                    - **القياس**: تكلفة اكتساب العميل، التحويل، الاحتفاظ، تكرار شراء.
                    - **بوابة قرار**: لا توسيع قناة إذا تكلفة اكتساب العميل أعلى من حد القبول.
                - 3) نطاق المنصة — تفصيلي
                    - **القرارات**: ترتيب الفئات حسب (التكرار × الهامش × التعقيد التنظيمي).
                    - **المخرجات**: مصفوفة حساسية + حزم تفعيل + قائمة متطلبات لكل فئة.
                    - **التشغيل**: تعريف مالك لكل فئة + دليل تشغيل الفئة.
                    - **الامتثال**: جدول متطلبات لكل فئة (عمر/وصفة/تبريد/مواد مقيدة).
                    - **القياس**: مؤشر أداء رئيسي لكل فئة + مساهمة ربحية مباشرة.
                    - **بوابة قرار**: فئة منظمة تتطلب (التحقق من الهوية والتحقق من الشريك التجاري + إثبات التسليم + تدقيق).
                - 4) ما نبنيه (واجهات المنتج) — تفصيلي
                    - **القرارات**: الحد الأدنى من المنتج القابل للإطلاق لكل واجهة + أولويات تجربة المستخدم + تصميم الأحداث.
                    - **المخرجات**:
                        - وثيقة متطلبات المنتج + مسارات تجربة المستخدم.
                        - تصنيف الأحداث (بحث/سلة/دفع/تسليم/نزاع).
                        - معايير قبول (معايير القبول).
                    - **التشغيل**: روتين إصدار أسبوعي + إطلاق تدريجي + خطة رجوع.
                    - **الامتثال**: متطلبات إشعارات/موافقة/إفصاحات بحسب الفئة.
                    - **القياس**: مسار التحويل + زمن إتمام الطلب + فشل الدفع.
                    - **بوابة قرار**: لا ميزة دون تهيئة القياس.
                - 5) القيمة المقترحة — تفصيلي
                    - **القرارات**: 4 رسائل قيمة منفصلة + عروض/ضمانات.
                    - **المخرجات**: صفحات قيمة + حزم عروض + سياسات استرجاع.
                    - **التشغيل**: مراجعة أسبوعية لشكاوى المستخدم وربطها بالقيمة.
                    - **الامتثال**: صياغة شروط الخدمة + سياسات الاسترجاع/الإلغاء.
                    - **القياس**: مؤشر الرضا الصافي ومؤشر رضا العميل + تكرار شراء + معدل الشكاوى.
                    - **بوابة قرار**: أي وعد قيمة يجب أن يقابله مؤشر أداء رئيسي.
                - 6) نموذج الشراكة — تفصيلي
                    - **القرارات**: نسبة العمولة + رسوم + شروط دفع + مستويات خدمة.
                    - **المخرجات**:
                        - اتفاقية شراكة (نسخة معيارية).
                        - دليل انضمام الشريك + قائمة تحقق.
                        - سياسة تسويات/مطالبات.
                    - **التشغيل**: التحقق من الشريك التجاري + ربط نظام المخزون/نقاط البيع + تدريب تجهيز.
                    - **الامتثال**: تحقق تراخيص/سجل تجاري/بيانات بنكية.
                    - **القياس**: صحة الشريك + دقة المخزون + زمن التجهيز.
                    - **بوابة قرار**: تعليق شريك عند تكرار فشل هدف مستوى الخدمة.
                - 7) نواة المنصة — تفصيلي
                    - **القرارات**: النطاقات + حدود المسؤوليات + واجهات برمجة التطبيقات
                    - **المخرجات**: خريطة النطاقات + عقود واجهات برمجة التطبيقات + سجل أحداث.
                    - **التشغيل**: إدارة إصدارات واجهة برمجة التطبيقات + توافق خلفي.
                    - **الامتثال**: التحكم في الوصول بحسب الدور + مسارات التدقيق + سياسات احتفاظ.
                    - **القياس**: زمن الاستجابة/التوافر/معدل الأخطاء لكل نطاق.
                    - **بوابة قرار**: لا نطاق دون أهداف مستوى الخدمة.
                - 8) التقنية — تفصيلي
                    - **القرارات**: الانتقال تدريجيًا من نظام موحد إلى خدمات تدريجي + ناقل أحداث + حزمة البيانات.
                    - **المخرجات**: سجلات قرارات معمارية + الملاحظة والتشخيص حزمة + التعافي من الكوارث دليل إجراءات.
                    - **التشغيل**: التكامل المستمر والنشر المستمر + مفاتيح تفعيل الخصائص + قطار الإصدارات.
                    - **الامتثال**: تشفير + إدارة الأسرار + فصل بيئات.
                    - **القياس**: أهداف مستوى الخدمة/هدف مستوى الخدمة + متوسط زمن الإصلاح + معدل الحوادث.
                    - **بوابة قرار**: لا توسع دون مراقبة سجلات/مقاييس/آثار تتبع.
                - 9) التشغيل — تفصيلي
                    - **القرارات**: نموذج نطاقات التشغيل + إدارة السعة + سياسات الإلغاء.
                    - **المخرجات**: إجراءات تشغيل معيارية (تجهيز/استبدال/توصيل/نزاع) + الحوادث دليل تشغيل.
                    - **التشغيل**: المناوبة + تصنيف شدة + تحليل السبب الجذري + إجراءات تصحيحية ووقائية.
                    - **الامتثال**: تتبع إثبات التسليم + سجلات جودة + ضبط سلاسل تبريد.
                    - **القياس**: الالتزام بالوقت، الإلغاء، تكلفة التوصيل، متوسط زمن الإصلاح.
                    - **بوابة قرار**: وقف توسع إذا تجاوزت الحوادث حدًا أسبوعيًا.
                - 10) الاقتصاديات والإيرادات — تفصيلي
                    - **القرارات**: نماذج رسوم + اشتراكات + إعلانات + أعمال إلى أعمال.
                    - **المخرجات**: نموذج اقتصاديات وحدة + لوحة مساهمة ربحية مباشرة.
                    - **التشغيل**: تجارب تسعير مضبوطة + مراجعة أسبوعية للهامش.
                    - **الامتثال**: شفافية الرسوم + فواتير/ضرائب حسب الدولة.
                    - **القياس**: هامش المساهمة لكل طلب + نسبة العمولة + معدل الاسترجاع.
                    - **بوابة قرار**: فئة خاسرة تُوقَف أو تُعاد هندستها.
                - 11) القياس (مؤشرات الأداء) — تفصيلي
                    - **القرارات**: قاموس المقاييس + مصدر الحقيقة.
                    - **المخرجات**: قاموس المقاييس + لوحات + تنبيهات حدود.
                    - **التشغيل**: مراجعة يومية تشغيلية + أسبوعية اقتصاديات.
                    - **الامتثال**: إخفاء الهوية للتحليلات + صلاحيات الوصول.
                    - **القياس**: اكتمال بيانات الأحداث + دقة القياس.
                    - **بوابة قرار**: قرار بدون بيانات = قرار مرفوض.
                - 12) الإطلاق العالمي — تفصيلي
                    - **القرارات**: ترتيب المدن/الدول + متطلبات محلية (لغة/عملة/ضرائب).
                    - **المخرجات**: حزمة المدينة + حزمة الدولة + قوائم التحقق.
                    - **التشغيل**: مرحلة تجربة أولية + توسع تدريجي + قرار الانطلاق أو الإيقاف بوابات.
                    - **الامتثال**: محرك سياسات قابل للتهيئة + وثائق تدقيق.
                    - **القياس**: الزمن حتى الإطلاق + الاستقرار بعد الإطلاق.
                    - **بوابة قرار**: لا دولة دون امتثال موثق واختبارات.
                - 13) إدارة المخاطر — تفصيلي
                    - **القرارات**: حدود الاحتيال/النزاعات/السمعة + خطة استجابة.
                    - **المخرجات**: سجل المخاطر + ضوابط + سيناريوهات فشل.
                    - **التشغيل**: مراقبة احتيال + مراجعة أسبوعية للحوادث.
                    - **الامتثال**: تدقيقات دورية + توثيق إجراءات تصحيحية ووقائية.
                    - **القياس**: معدل الاحتيال + معدل النزاعات + معدل الاسترجاع الإجباري.
                    - **بوابة قرار**: مفتاح إيقاف طارئ عند تجاوز الحدود.
                - 14) العقائد — تفصيلي
                    - **القرارات**: تحويل كل عقيدة إلى سياسة + مؤشر أداء رئيسي.
                    - **المخرجات**: سياسات مكتوبة + اختبارات امتثال داخلية.
                    - **التشغيل**: مراجعة شهرية التزام.
                    - **القياس**: درجة التدقيق + انحرافات.
                - 15) آلية التحسين — تفصيلي
                    - **المخرجات**: سجل تجارب + قالب قرار + سجل تغييرات.
                    - **التشغيل**: دورة أسبوعية (فرضية→تجربة→نتيجة→قرار).
                    - **بوابة قرار**: كل تغيير له خطة رجوع.
                - 16) الثغرات وكيف نغلقها — تفصيلي
                    - **المخرجات**: مصفوفة الثغرات + ملاك + تواريخ.
                    - **القياس**: زمن الإغلاق + أثر الإغلاق على مؤشر أداء رئيسي.
                - 17) التحسينات الحاسمة — تفصيلي
                    - **المخرجات**: قائمة الأعمال استراتيجي مرتبط ببوابات التوسع.
                    - **التشغيل**: ترتيب أولويات شهري.
                - 18) السجلات الاستراتيجية — تفصيلي
                    - **المخرجات**: المخاطر والامتثال والحوادث والجودة والتجارب والأهداف والنتائج الرئيسية.
                    - **التشغيل**: تحديثات ثابتة + قابلية تدقيق.
                - 19) معجم المصطلحات — تفصيلي
                    - **المخرجات**: قاموس مصطلحات + تعريفات مؤشرات أداء رئيسية + قواعد تسمية.
                    - **بوابة قرار**: التزام المصطلحات في كل الوثائق/المنتج.
                    - **القياس**: نسبة التوحيد (بدون مرادفات متضاربة).
    - ما الذي نُقدمه (قيمة عالمية)
        - تجربة موحدة للمستخدم: بحث، دفع، تتبع، دعم.
        - تمكين الشركاء: أدوات كتالوج ومخزون وتسعير وتجهيز وتسوية.
        - تشغيل قوي للفئات الحساسة: تبريد، حمولة، طاقة، أدوية، خدمات ميدانية، وفق الأنظمة.
        - حوكمة قابلة للتدقيق: سجلات أحداث، إثبات تسليم، ومسارات نزاعات واضحة.
    - لماذا ننجح عالميًا (عوامل التميز)
        - نواة منصة واحدة قابلة للتوسع، مع سياسات قابلة للتهيئة حسب الدولة والمدينة.
        - فصل واضح بين “المنتج” و“التشغيل” و“الامتثال” بحيث يمكن التوسع دون فوضى.
        - تصميم يعتمد على البيانات: قياس لحظي للجودة والوقت والتكلفة والإيراد لكل فئة.
        - ثقة وسلامة: مكافحة احتيال، ضوابط مدفوعات، إدارة نزاعات، وحماية المستخدم.
    - ما الذي سنبنيه (مخرجات ملموسة)
        - تطبيق المستهلك.
        - لوحة الشريك.
        - تطبيق القوى العاملة.
        - لوحة تشغيل مركزية.
        - بوابة تكاملات للشركاء والجهات.
    - نموذج التشغيل العالمي (مختصر)
        - نطاقات تشغيل، نوافذ زمنية، وتخطيط سعة.
        - إسناد ذكي وتجميع مهام وتحسين مسارات.
        - معايير جودة وتدقيق دوري للشركاء.
        - إدارة حوادث واستجابة حسب شدة الحادث.
    - الامتثال والخصوصية (مختصر)
        - محرك سياسات يضبط ما يُسمح به حسب الدولة والفئة.
        - تحقق من الهوية والتحقق من الشريك التجاري بحسب متطلبات الأنظمة.
        - حوكمة بيانات: تحكم في الوصول، تشفير، احتفاظ بيانات، موافقة المستخدم، حق الحذف ضمن الاستثناءات النظامية.
    - الاقتصاديات والربحية (مختصر)
        - عمولات ورسوم توصيل ورسوم فئات خاصة.
        - اشتراكات للشركاء وللعائلات.
        - إعلانات مدفوعة داخل المنصة.
        - عقود أعمال إلى أعمال.
        - القياس الأساسي: اقتصاديات الوحدة ومساهمة ربحية مباشرة لكل فئة.
    - مؤشرات نجاح موحدة (تُقاس لكل فئة)
        - الالتزام بالوقت.
        - نجاح المدفوعات.
        - نسبة الإلغاء.
        - مؤشر رضا العميل.
        - جودة الشركاء وصحة الشريك.
    - خطة إطلاق عالمية (مراحل)
        - تجربة في مدينة واحدة بفئتين.
        - توسع داخل المدينة حتى سبع فئات.
        - مدينة ثانية داخل الدولة.
        - دولة ثانية عبر تهيئة السياسات والمدفوعات والضرائب.
        - بوابات قرار تمنع التوسع قبل تحقق الاعتمادية والاقتصاد والثقة والامتثال.
